# Mbio Search

多源学术文献检索技能包，集成四大数据源，一站式满足文献调研全流程需求。

## 数据源

| 数据源 | 覆盖领域 | 数据量 |
|--------|----------|--------|
| **PubMed** | 生物医学、生命科学 | 3700万+ 篇 |
| **CrossRef** | 全学科（DOI注册机构） | 1.5亿+ 元数据 |
| **arXiv** | 物理、数学、计算机、生物学预印本 | 240万+ 篇 |
| **Google Scholar** | 综合学术（含专利、图书、学位论文） | 约2亿+ 条记录 |

### 🔍 多源并发搜索
同时查询 CrossRef / PubMed / arXiv / Google Scholar 四大数据源，自动合并去重，按相关度/日期/引用数智能排序，单次请求覆盖全学科。

### 📄 按 ID 获取详情
支持 DOI、PMID、arXiv ID 自动识别与解析，智能补全标题、作者、期刊、摘要、引用数等完整元数据，返回可直接访问的全文链接（优先开放获取）。

### 📝 格式化引用
内置 APA / Nature / IEEE / Vancouver / MLA 等多种学术引用风格，一键生成标准格式参考文献条目，支持单篇或批量导出。

### 🎯 MeSH 词表辅助
基于 PubMed 官方 MeSH 词表，辅助构建精准检索式，自动扩展同义词、下钻主题词，提升文献查全率与查准率。

### 🔄 文献管理脚本
支持 .nbib / .ris / .bib / .enw 格式互转，无缝对接 Zotero、EndNote、Mendeley 等主流文献管理软件，批量重命名、去重、字段清洗一站完成。
